Monier Williams Sanskrit English Dictionary

Based on the Data Available at the University of Cologne

The Monier Williams Sanskrit Dictionary is an Ajax based Java Software.

Special Features
  • Ability to Search with partial Input ( left, right, middle portions of Texts )
  • Auto-completion and auto-suggest - as yet unimplemented
  • Choice of Input in SLP/ITRANS/HK
  • Simultaneous view of Devanagari as User Keys in Roman
