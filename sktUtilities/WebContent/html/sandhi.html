Sandhi Engine

The Sandhi Engine Software - संधि विवेक  - simulates the rules of Sandhi - संधि - as discussed by Varadaraja - वरदराज - in the Laghu Siddhanta Kaumudi - लघु सिद्धान्त कौमुदी  -. Effort has been taken to incorporate all rules even those which are applicable to only one unique pair of words( for example aksha + uhinI = akshauhiNI not akshohiNI) amongst the innumberable forms possible.

However, rules which make Coding difficult especially ones which apply to all inflected forms of a Noun or Verb are left out as this is a Sandhi Simulator not a Declension Engine.

In the preparation of this Work I was hugely dependent upon Swami Dayanada Saraswati's 'Sandhi Niyma' and Vaidya Bhimsena Shastri's Hindi Translation and Explanation of the LSK. I also consulted Prof. Ram Sharma's English Translation of the Ashtadhyayi along with the Hindi translation of Brahma Dutaa Jijnasu. Without these books especially the Bhaimi Vyakhya - which is a mine of lucidity and comprehenive explanations - , I would not have been able to easily create this Software.

Useful Tips

(1). When a Consonant Sandhi Paradigm is being entered, always give another try by checking the ‘padanta’ check-box. There are a lot of examples where the Output changes if you have checked ‘padanta.

(2). If sometimes you don't see any changes in the Output from the Original but Sutras have been Quoted, you should remember the Dictum that Scriptures are like Rain ( shastraH parjanyavat ) which falls without consideration on where it is falling.

(3). If you get confused by the use of pratyaharas like 'jash', 'khar' etc and want to know which phonemes they consist of, use 'Find Pratyahara' and simply enter the pratyahaara abbreviation in Encoding of your choice and the results will be displayed.

(4). Sutra Numbers may vary because I have exactly reproduced used the Sutra Index quoted in the Book I happened to be referring at the time. Pls. report an error in this regard only if the Sutra seems to be wide off the mark.

(5) As yet their is no support for Reduplication( 'anaci ca' etc ) and certain kinds of Elision('jharo jhari savarNe). I hope to incorporate these in Future Releases.
